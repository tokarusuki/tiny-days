# Tiny Days

Tiny Days is a cozy, Tamagotchi-inspired browser game for leisurely play. You care for a tiny creature throughout soft little days: feeding it, playing gently, organizing its space, and observing small changes over time.

There is no game over. Your companion does not die—it feels, reacts, and grows alongside you.

## Play

Open `index.html` in a modern browser and spend a quiet minute with your tiny friend.

## Features

- ASCII companion art that changes with its mood.
- Gentle care actions: feed, play, tidy, and observe.
- A tiny journal of recent moments.
- Slow day-by-day stat changes that invite return visits without pressure.
